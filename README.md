# 🧠 BodyCapture AI – Real-Time Pose Detection System

## 📌 Problem Statement

In many real-world applications like fitness tracking, healthcare monitoring, and gesture-based control systems, accurately detecting human body movements in real-time is challenging. Traditional systems require expensive hardware or lack precision and scalability.

---

## 💡 Solution

**BodyCapture AI** is a real-time human pose detection system that uses advanced machine learning models to track and analyze body movements through a webcam. It enables accurate and efficient pose estimation without the need for specialized hardware.

---

## 🚀 Features

* 🎯 Real-time body pose detection
* 📸 Webcam-based motion tracking
* 🤖 AI-powered pose estimation (PoseNet / ML5.js)
* ⚡ Fast and lightweight performance
* 🎮 Can be extended for gesture control & fitness apps

---

## 🛠 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Machine Learning:** ML5.js, PoseNet
* **Libraries:** p5.js
* **Platform:** Web-based

---

## 🧩 How It Works

1. Captures live video using webcam
2. Applies PoseNet model to detect key body points
3. Maps joints like nose, eyes, shoulders, etc.
4. Displays skeleton structure in real-time
